# 세대 8 이야기 — "가장 큰 비밀을 알아냈어요" 🔑

> 칩 종류를 4개로 늘리고, 점쟁이 점수도 역대 최고였어요. 그런데도 비겼어요. 그리고 5세대 내내 본 가장 중요한 사실이 또렷해졌어요.

## 🎯 알고 싶었던 것
우리는 그동안 **세 가지를 모두 고쳤어요**: ① 심판을 고치고(세대 6), ② 안내문을 또렷이 하고(세대 6),
③ 새 칩 `jpeg`를 추가해서 시험 종류를 4개로 늘렸어요(세대 8). **이 셋을 다 갖추면 드디어 새 챔피언이 나올까?**

## 🧪 우리가 한 일
- 칩 **4종류**로 시험: `gcd` + `aes` + `ibex` + `jpeg` (총 7,194개 문제!).
- 1등 도전자는 빗나감 **0.53** — **지금까지 중 가장 똑똑한** 점수였어요!
- 깜짝 시험(LODO)과 고친 T1 심판으로 채점했어요.

## 📊 결과 (쉬운 말로)
- 깜짝 시험: 4종류 중 **2종류 승, 2종류 패** → "비김(mixed)"
- T1 심판: 도전자와 챔피언이 거의 똑같음 (2.24 vs 2.22) → **"비겼다"** → 탈락 (rejected_t1)
- 재미있는 점: 도전자는 `jpeg`·`aes` 칩은 잘 맞혔는데(이 칩들이 서로 비슷해서요), `gcd`·`ibex`는 못 맞혔어요.

## 💡 이 실험이 가르쳐준 것 (가장 중요!)
**점쟁이 점수(val_mae)는 계속 좋아지는데(1.29 → 0.53), "처음 보는 칩에 강한 것"은 안 늘어요.**
세대 4부터 8까지 **다섯 번 내내** 같은 사실을 봤어요:

> ### 🌟 "시험 점수를 잘 받는 것"과 "처음 보는 문제에 강한 것"은 완전히 다른 능력이에요!

이건 우리 프로젝트가 발견한 **가장 큰 비밀**이에요. 그냥 점수만 높이거나 데이터를 늘리는 것으로는
이 벽을 못 넘어요. 그리고 — 만약 심판이 없었다면, 점수만 보고 이 "가짜 우등생"을 5번이나 챔피언으로
뽑았을 거예요. **심판들이 매번 막아준 덕분에 챔피언이 망가지지 않았어요.**

## ⏭️ 다음 이야기
이제 우리는 알아요: 단순히 더 열심히 하는 걸로는 안 돼요. **완전히 새로운 방법**이 필요해요 —
예를 들어 "칩 종류에 상관없이 통하는 비결"을 점쟁이가 배우게 하는 것. 그게 다음 모험이에요!

## 💬 어려운 말 풀이
- **mixed(비김)**: 깜짝 시험에서 이긴 칩과 진 칩 수가 비슷한 것.
- **jpeg**: 가장 큰 칩(문제 4,410개). 너무 커서 다른 칩들보다 영향력이 셌어요.
- **가짜 우등생**: 연습 문제는 잘 푸는데 처음 보는 문제는 못 푸는 학생. 우리 도전자들이 자꾸 이랬어요.
