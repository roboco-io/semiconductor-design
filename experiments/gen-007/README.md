# 세대 7 이야기 — "누가 이겼나"와 "확실히 이겼나"는 다른 질문 🤔

> 고친 심판으로 첫 시험. 도전자가 깜짝 시험은 통과했지만, "그건 운일 수도 있어"라며 심판이 막았어요.

## 🎯 알고 싶었던 것
세대 6에서 심판(T1)을 고쳤어요. **이제 고친 심판으로 돌리면 새 챔피언이 나올까?**

## 🧪 우리가 한 일
- 고친 심판(교차설계 T1)으로 도전자들을 채점했어요.
- 1등 도전자는 빗나감 **1.29**로 지금까지 중 제일 똑똑해 보였어요!
- 깜짝 시험(LODO)도 통과했어요 (3종류 중 2종류에서 이김).

## 📊 결과 (쉬운 말로)
- 🎉 깜짝 시험: **통과** (2/3 승)
- ❌ T1 심판: **"비겼다 (확실하지 않아)"** → 탈락 (rejected_t1)
- 왜냐면 도전자가 `aes`·`gcd` 칩은 이겼지만 **`ibex` 칩에서 크게 졌거든요**. 평균을 내보니
  "확실히 이겼다"고 말하기 어려웠어요. 심판이 "그 정도 차이는 운일 수 있어"라고 한 거예요.

## 💡 이 실험이 가르쳐준 것
**"누가 더 많이 이겼나"와 "확실히 이겼나"는 다른 질문이에요!**
- 깜짝 시험(LODO)은 "**몇 종류 칩에서 이겼나**"를 세요 (2종류 > 1종류니까 통과).
- T1 심판은 "**그게 운이 아니라 확실한가**"를 수학으로 따져요 (한 칩에서 크게 지면 확실하지 않다고 봐요).
- 두 심판이 **서로 다른 걸 보면서 함께** 가짜 우승자를 걸러내는 거예요. 팀워크죠!

## 💬 어려운 말 풀이
- **교차설계 T1**: 고친 심판. 깜짝 시험과 똑같이 "처음 보는 칩"으로 재되, "확실한가"를 통계로 따져요.
- **indistinguishable(비김)**: 도전자와 챔피언이 통계적으로 구별 안 됨. 확실한 승부가 안 났다는 뜻.
- **ibex**: 다른 칩들과 성격이 아주 다른(숫자가 훨씬 큰) 칩. 자꾸 말썽을 일으켜요.
