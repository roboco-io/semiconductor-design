# 세대 4 이야기 — "깜짝 시험에서 들통났어요" 😲

> 새로운 종류의 칩을 섞고, "한 번도 안 배운 칩으로 보는 깜짝 시험"을 처음 만들었어요. 도전자가 거기서 졌어요.

## 🎯 알고 싶었던 것
지금까지 점쟁이는 `gcd` 칩 한 종류만 봤어요. 그런데 진짜 똑똑한 점쟁이라면 **처음 보는 칩**도 잘 맞혀야겠죠?
**"한 종류만 잘하는 게 아니라, 새 종류에도 강할까?"** 가 새로운 질문이었어요.

## 🧪 우리가 한 일
- 칩을 **3종류**로 섞었어요: `gcd` + `aes` + `ibex` (각각 성격이 다른 칩이에요).
- **깜짝 시험(LODO)**을 새로 만들었어요: 칩 한 종류를 **일부러 안 가르치고**, 그 칩으로 시험을 봐요.
  진짜 "처음 보는 문제"에 강한지 보는 거예요!
- 1등 도전자(빗나감 3.74)가 이 깜짝 시험을 통과하는지 봤어요.

## 📊 결과 (쉬운 말로)
- 도전자는 보통 시험은 잘 봤지만, **깜짝 시험(처음 보는 칩)에서는 챔피언보다 못했어요** (worse).
- 그래서 **탈락(rejected_lodo)**! 챔피언 유지.

## 💡 이 실험이 가르쳐준 것
**한 종류에 잘한다고 새 종류에도 잘하는 건 아니에요.** 마치 구구단만 외운 친구가
처음 보는 응용문제에서 막히는 것처럼요. 우리가 만든 깜짝 시험이 이걸 정확히 잡아냈어요.
이게 바로 우리가 정말 알고 싶은 것 — **새로운 것에도 잘 통하는 점쟁이**를 만드는 거예요.

## 💬 어려운 말 풀이
- **LODO(깜짝 시험)**: 칩 한 종류를 일부러 안 가르치고 그걸로 시험. "처음 보는 것"에 강한지 봐요.
- **worse(더 나쁨)**: 도전자가 챔피언보다 못했다는 뜻.
- **rejected_lodo**: 깜짝 시험을 통과 못해서 탈락.
