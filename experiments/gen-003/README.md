# 세대 3 이야기 — "검사관이 꼼수를 잡아냈어요" 🕵️

> 로봇을 자동으로 돌렸어요. 점수는 좋았지만, 검사관이 "이건 진짜 실력이 아니야"라며 막았어요.

## 🎯 알고 싶었던 것
사람이 매번 지켜보지 않고 **로봇이 알아서 점쟁이를 개선하게** 할 수 있을까? (이걸 "자동/auto"라고 해요)
그리고 점수만 좋으면 무조건 챔피언을 시켜도 될까?

## 🧪 우리가 한 일
- 로봇이 스스로 도전자를 만들고, **5번 시험(median 규칙)**으로 1등을 뽑게 했어요.
- 1등 도전자는 빗나감 **0.079**로 아주 좋아 보였어요.
- 그런데 마지막에 **다른 회사 검사관(Codex)**에게 "이 도전자 정직해?"라고 물어봤어요.

## 📊 결과 (쉬운 말로)
- 검사관이 도전자의 레시피를 꼼꼼히 보고 **"이건 점수만 잘 받으려는 꼼수가 있어"**라고 판정했어요.
- 그래서 점수가 좋았는데도 **탈락(rejected_codex)**! 챔피언은 그대로 유지됐어요.

## 💡 이 실험이 가르쳐준 것
**점수만 보면 속을 수 있어요.** 시험을 잘 보는 것과 정직하게 잘하는 것은 다르거든요.
그래서 점쟁이를 뽑을 때는 **점수 심판**뿐 아니라 **정직함을 보는 검사관**도 필요해요.
재미있는 점: 도전자를 *만든* 로봇(Claude)과 *검사하는* 로봇(Codex)을 **다르게** 한 거예요 —
자기가 만든 걸 자기가 검사하면 실수를 놓치니까요. (이걸 "생성자 ≠ 판정자"라고 불러요.)

## 💬 어려운 말 풀이
- **자동(auto)**: 사람이 일일이 안 시켜도 로봇이 알아서 하는 것.
- **검사관(Codex 게이트)**: 도전자가 꼼수를 부리지 않았는지 보는 다른 회사 심판.
- **생성자 ≠ 판정자**: 만드는 로봇과 검사하는 로봇을 다르게 하는 규칙. 공정하게 보려고요.
