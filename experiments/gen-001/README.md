# 세대 1 이야기 — "로봇이 사람 챔피언을 이겼어요!" 🏆

> 이 프로젝트의 첫 실험. AI 로봇이 스스로 더 똑똑한 "점쟁이 기계"를 만들어 사람이 만든 챔피언을 이겼습니다.

## 🎯 알고 싶었던 것
우리는 **칩(컴퓨터 부품)이 다 만들어지기 전에 "이 칩이 빠르게 잘 작동할까?"를 미리 맞히는 점쟁이 기계**를
만들고 있어요. 이 점쟁이를 사람이 가르치는 것보다 **AI 로봇이 더 잘 가르칠 수 있을까?** 가 첫 질문이었어요.

## 🧪 우리가 한 일
- 로봇 요리사 2명(Claude, Codex)에게 "점쟁이를 가르치는 레시피"(`train.py`)를 주고 **맘대로 바꿔보라**고 했어요.
- 바꾼 레시피로 점쟁이를 훈련시키고, 예측이 얼마나 빗나가는지(이걸 **val_mae**라고 해요, 작을수록 똑똑함) 쟀어요.
- 사용한 칩은 `gcd`라는 한 종류였어요(53개의 시험 문제).

## 📊 결과 (쉬운 말로)
- 사람이 만든 챔피언 점쟁이: 빗나감 **0.18**
- Codex 로봇이 만든 점쟁이: 빗나감 **0.10** → **더 똑똑함!** 🎉
- 그냥 아무 생각 없이 찍는 방법(naive)보다는 **13배** 정확했어요.
- 그래서 이 로봇 점쟁이가 **새 챔피언으로 등극**했어요 (이걸 "승격/promoted"이라고 해요).

## 💡 이 실험이 가르쳐준 것
**AI 로봇이 사람보다 더 좋은 방법을 스스로 찾아낼 수 있다**는 걸 처음 확인했어요.
단순히 숫자만 살짝 바꾼 게 아니라, 로봇이 *새로운 아이디어*(여러 점쟁이의 의견을 합치는 방법)를 떠올린 거예요.

## 💬 어려운 말 풀이
- **점쟁이 기계(surrogate)**: 칩을 끝까지 안 만들어도 성능을 미리 맞히는 AI.
- **val_mae**: 예측이 실제와 얼마나 빗나갔는지. 작을수록 똑똑함.
- **챔피언(baseline)**: 지금 1등인 점쟁이. 새 도전자는 이걸 이겨야 해요.
- **승격(promoted)**: 도전자가 챔피언을 이겨서 새 1등이 되는 것.
